Differential Geometry¶

This is an introduction to differential geometry based on MATH 147A at UC Santa Barbara. We will discuss curves and surfaces.

Parametrized Curves¶

Let $\gamma: I\to \mathbb{R}^3$, where $I$ is an interval in $\mathbb{R}$. We say that $\gamma$ is a parametrized curve. let $t\in I$. Then $\gamma(t)=((x(t),y(t),z(t))$ is a point on the curve.

In [1]:
import numpy as np
import matplotlib.pyplot as plt

# Define the parameter t
t = np.linspace(0, 2 * np.pi, 1000)

# Define the parametric equations for x(t) and y(t)
x = np.sin(t) + np.sin(3*t)/2
y = np.cos(t) - np.cos(2*t)/2

# Plot the curve
plt.figure(figsize=(8, 6))
plt.plot(x, y)
plt.title("Parametrized Curve")
plt.xlabel("x(t)")
plt.ylabel("y(t)")
plt.grid(True)
plt.axis('equal')
plt.show()

The example above is a curve in the plane and it is not really in a regular shape. Let's consider some familiar examples below:

Constant Curves and Straight Lines¶

A contant curve or a point is simply given by $$ \gamma(t)=P_0 $$for a fixed $P_0\in\mathbb{R}^3$ A straight line is given by $$ \gamma(t)=P_0+t\vec{v}, t\in\mathbb{R} $$ where $\vec{v}$ is a direction vector.

# Define the parameter t for the line
t = np.linspace(-5, 5, 1000)
# Point P coordinates
Px, Py, Pz = 1, 2, 3
# Line Q coordinates and direction vector
Qx, Qy, Qz = 0, 0, 0
a, b, c = 1, 1, 1
# Parametric equations for the line
x = Qx + a * t
y = Qy + b * t
z = Qz + c * t
# Create a 3D plot
fig = plt.figure(figsize=(8, 6))
ax = fig.add_subplot(111, projection='3d')
# Plot the point P
ax.scatter(Px, Py, Pz, color='red', s=100, label='Point P(1,2,3)')
# Plot the line
ax.plot(x, y, z, label='Line through Q(0,0,0) with direction (1,1,1)')
ax.set_title("Point and Line in R^3")
ax.set_xlabel("x")
ax.set_ylabel("y")
ax.set_zlabel("z")
ax.grid(True)
ax.legend()
plt.show()

Tangent Vectors¶

Let $\gamma: I \to \mathbb{R}^3$ be a parametrized curve and let $t_0\in I$. Define the tangent vector $\gamma'(t_0)$ at point $t_0$ to be $$ \gamma'(t_0)=\lim_{t\to t_0} \frac{\gamma(t)-\gamma(t_0) }{t-t_0} $$ The intuition from calculus should be clear in this case.

import numpy as np
import matplotlib.pyplot as plt

# Define the parameter t for the circle
t = np.linspace(0, 2 * np.pi, 1000)

# Circle parameters
R = 1
x = R * np.cos(t)
y = R * np.sin(t)

# Tangent vector at t = pi/4
tangent_point = 3*np.pi/2
Tx = -R * np.sin(tangent_point)
Ty = R * np.cos(tangent_point)

# Plot the circle
plt.figure(figsize=(8, 6))
plt.plot(x, y, label='Circle')
plt.quiver(R * np.cos(tangent_point), R * np.sin(tangent_point), Tx, Ty, angles='xy', scale_units='xy', scale=1, color='r', label='Tangent Vector')
plt.title("Circle with Tangent Vector at t = 3π/4")
plt.xlabel("x")
plt.ylabel("y")
plt.grid(True)
plt.axis('equal')
plt.legend()
plt.show()
